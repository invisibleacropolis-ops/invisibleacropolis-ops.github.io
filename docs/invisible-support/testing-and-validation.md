# Testing and validation

## Recommended command checks
- `npm run generate:pages`
- `npm run build`

## Manual panel validation
- Upload workflow renders and accepts valid files.
- Repository storage settings persist and reconnect.
- Asset library and Document viewer stay synchronized.
- Image upload, Image gallery, and Image viewer stay synchronized.
- Storage meter updates after upload/delete operations.

## Reporting format for outside engineers
For each run, record:
- command,
- timestamp,
- pass/fail status,
- environment constraints,
- linked evidence (logs or screenshots).
