# Glossary

- **Panel**: A distinct Home UI section anchored by a heading ID in `index.html`.
- **Feature module**: JS module under `public/InvisibleSupport/src/features/**` implementing panel behavior.
- **Store**: State manager module responsible for persistence and update propagation.
- **Manifest**: JSON metadata list stored in `public/InvisibleSupport/storage/*.json`.
- **Shared systems**: Reusable infrastructure/services consumed by multiple feature modules.
